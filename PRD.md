# 產品需求文件（PRD）

## 專案名稱
奧修禪卡線上占卜服務網站

## 專案背景
奧修禪卡是一款結合靈性與現實指引的占卜卡牌，強調透過當下片刻的反思與覺察，幫助用戶內省、啟發心靈。提供一個簡易、直觀的線上占卜平台，讓用戶能快速抽取奧修禪卡牌，並獲得卡牌的靈性解讀與指引。服務將結合 Python Flask 技術，實現前後端整合，達成輕量且易維護的系統。

## 專案目標
- 建立一個簡單易用的奧修禪卡線上占卜網站
- 實現隨機抽取奧修禪卡牌的功能
- 提供每張卡牌的詳細解釋與靈性指引文字說明
- 支援良好的使用者介面體驗與流暢的占卜流程
- 採用 Python Flask 技術完成全站開發，便於後續擴充和維護

## 目標用戶
- 對靈性占卜有興趣的初學者與愛好者
- 希望透過簡單工具自我探索與心靈覺察的使用者

## 使用者流程（User Journey）
1. 使用者進入首頁，閱讀簡介後點擊「開始占卜」。
2. 跳轉占卜頁面，使用者點擊「抽卡」按鈕。
3. 系統隨機抽取一張奧修禪卡，並顯示卡牌及說明。
4. 使用者可選擇查看詳細解釋或保存結果，並可繼續抽卡。
5. 使用者能在「歷史紀錄」中查看之前的抽卡結果。

## 系統架構概要
- 前端使用 Flask 的 Jinja 模板渲染頁面，提供響應式 UI。
- 後端由 Flask 處理抽卡邏輯與卡牌資料管理。
- 卡牌資料儲存在伺服器靜態 JSON 或簡易資料庫中。
- 支持 Session 管理用於歷史紀錄保存。
